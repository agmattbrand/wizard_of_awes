wizard_of_awes by agmattbrand

A simple gem for managing small snippets of HTML to be inserted into your application.

This gem comes with one tag woa_snippet and a simple CRUD admin for the snippets themselves. Inserting the tag on the page will insert the associated snippet. The tag supports a single optional parameter which will be used as a key to find the right snippet. If that key isn't provided, the full path of the current page will be used.

Dependencies

rails 3.1.0 or higher sass-rails formtastic client_side_validations

Install

gem install wizard_of_awes rails generate wizard_of_awes:install bundle exec rake db:migrate The install will create a wizard_of_awes initializer. There are 2 configurable properties including the ability to change the default route prefix ('woa-snippets'). Insert the wizard_of_awes tag in your views where you'd like to insert snippets: <%= woa_snippet %> You can specify an optional key if you don't want the current page's full path to be used as the key: <%= woa_snippet('my_key') %> You should also go to the CRUD admin and create your snippets: http://localhost:3000/woa-snippets/snippets
